This project was made in mainly Html5 assisted by CSS, JavaScript and PHP.

Html5 source - HTML stands for Hyper Text Markup Language. HTML uses elements to help the browsers display the content formed by the other codes.

Css source - Css is short for Cascading Style Sheets. Css is used for presenting documents that were written using html or xml. Css is the main form of code used for web browsers. We used the Css to format our website.

PHP source. - The acronym PHP is actually recursive which means it does not follow the typical format. PHP used to stand for Personal Homepage, but today it stands for hypertext preprocessor. PHP is also used for making web browsers just like Css. PHP runs on the browser and runs lines of commands.

JavaScript source - JavaScript is also used for making websites and it runs directly on the browser. JavaScript is used to make websites more appealing. Pop ups, drop down menus, videos, and more are all created using Javascript.
